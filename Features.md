## Features

Real-time ECG signal acquisition
Noise filtering for improved signal quality
Live ECG display on OLED
ECG data storage using SD card
Low-cost and portable design
Arduino Uno based implementation
Easy hardware integration
Suitable for educational and research applications
Low power consumption
Basic heart health monitoring

## Components 

Arduino Uno
AD8232 ECG Sensor Module
ECG Electrodes
OLED Display
Micro SD Card Module
Micro SD Card(2GB)
